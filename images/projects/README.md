# Project Images

Place your project images here.

## Folder Structure

- `project1-portrait.jpg` - Square portrait/thumbnail for project 1
- `project1-image1.jpg` - Detail image 1 for project 1
- `project1-image2.jpg` - Detail image 2 for project 1
- etc.

## Image Guidelines

- **Portraits**: Square images (1:1 aspect ratio), recommended 400x400px or larger
- **Detail images**: 16:9 aspect ratio recommended, any size works
- **Formats**: JPG, PNG, or WebP all work fine
- **Naming**: Use descriptive names like `project1-portrait.jpg`, `project2-image1.jpg`, etc.

## How to Add Images

1. Add your images to this folder
2. Update `projects.json` with the correct file paths
3. Use relative paths like `./images/projects/your-image.jpg`
